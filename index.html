My First Paragraph
Git is an open-source, distributed version control system that tracks changes to files. It operates on your local machine and creates a complete history of your code. GitHub is the cloud-based platform that hosts Git repositories. It provides a user-friendly interface and adds collaboration features on top of Git's functionality.
